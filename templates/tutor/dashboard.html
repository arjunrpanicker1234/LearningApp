Tutor Guide
Getting Started

As a tutor, you can create skills and upload learning resources for students.

Adding Skills

Click on "Add Skill" to create a new skill category. Each skill should have a name and description.

Uploading Resources

Resources can be text content or PDF files. For each resource, specify:

  • The skill it belongs to
  • A descriptive title
  • The proficiency level (1-5)
  • Content type (text or PDF)
Proficiency Levels

Use the following guidelines for proficiency levels:

  1. Beginner - No prior knowledge required
  2. Elementary - Basic concepts covered
  3. Intermediate - Requires some prior knowledge
  4. Advanced - Deep understanding of the subject
  5. Expert - Complex and specialized content
